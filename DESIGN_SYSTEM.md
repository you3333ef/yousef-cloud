# نظام التصميم - المنصة السورية للتطوير

## نظرة عامة
نظام تصميم شامل يدمج الهوية البصرية السورية مع أفضل ممارسات UI/UX الحديثة.

## الألوان الأساسية

### الألوان الوطنية السورية
```css
الأخضر السوري (Orontes Green): #054239
- الاستخدام: العناوين الرئيسية، العناصر التفاعلية
- الدلالة: الخصوبة والنمو في وادي العاصي

الذهبي (Qasioun Gold): #b9a779
- الاستخدام: الأزرار، التمييزات، العناصر المميزة
- الدلالة: غروب جبل قاسيون الذهبي

الرملية (Palmyra Sand): #edebe0
- الاستخدام: الخلفيات الفاتحة، البطاقات
- الدلالة: آثار تدمر التاريخية
```

### لوحة الألوان التفصيلية
```css
--syrian-primary: #054239           /* أخضر أساسي */
--syrian-primary-light: #428177     /* أخضر فاتح */
--syrian-primary-dark: #002623      /* أخضر داكن */

--syrian-secondary: #b9a779         /* ذهبي أساسي */
--syrian-secondary-light: #edebe0   /* رملية فاتحة */
--syrian-secondary-dark: #988561    /* ذهبي داكن */

--syrian-accent: #6b1f2a            /* أحمر قرمزي */
--syrian-neutral-light: #ffffff     /* أبيض */
--syrian-neutral-dark: #3d3a3b      /* رمادي داكن */
```

## الطباعة

### الخطوط العربية
```css
الأساسي: 'Qamra', 'Amiri', 'Tahoma', 'Arial', 'sans-serif'
البديل: 'Tahoma', 'Arial', 'sans-serif'
```

### خصائص الطباعة
- اتجاه النص: RTL (من اليمين إلى اليسار)
- حجم الخط الأساسي: 16px
- خط العناوين: Bold, متدرج من #054239 إلى #428177
- للمحرر: 'Fira Code', 'Consolas', 'Monaco', 'monospace'

## الرموز والأيقونات

### العقاب الذهبي السوري
- الوصف: شعار المنصة الرئيسي
- العناصر:
  - 14 ريشة جناح (المحافظات السورية)
  - 5 ريشات ذيل (المناطق الجغرافية)
  - 3 نجوم (الحرية والكرامة والاستمرارية)
- التدرجات: ذهبي (#b9a779) وأخضر (#054239)

### النقوش الهندسية
- نمط دائري مستوحى من الفسيفساء السورية
- شفافية: 5%
- استخدام: خلفيات فرعية

## المعالم السورية

تم تضمين رموز للمعالم التالية:
1. قلعة حلب - قلعة تاريخية شامخة
2. جبل قاسيون - رمز دمشق العريق
3. تدمر - مدينة النفط الفرعونية
4. الجامع الأموي - تحفة العمارة الإسلامية

## الاستجابة (Responsive Design)

### نقاط التوقف
- sm: 640px
- md: 768px
- lg: 1024px
- xl: 1280px
- 2xl: 1536px

### قواعد الاستجابة
- الشريط الجانبي: يخفي على الهواتف (< 1024px)
- البطاقات: 1 عمود على الهواتف، 2 على التابلت، 4 على الشاشات الكبيرة
- الشبكة: تتكيف تلقائياً مع حجم الشاشة

## إمكانية الوصول (Accessibility)

### التباين
- نسبة التباين النص/الخلفية: 4.5:1 على الأقل
- العناوين: 7:1
- العناصر التفاعلية: 3:1

### لوحة المفاتيح
- جميع العناصر التفاعلية تدعم التنقل بلوحة المفاتيح
- مؤشر واضح للتركيز (focus ring)

### قارئات الشاشة
- استخدام aria-labels للأيقونات
- عناوين واضحة ومنطقية (h1, h2, h3)
- alt text للصور

## أفضل الممارسات

1. **الاتساق**: استخدم نفس الزوايا والألوان والظلال في جميع العناصر
2. **البساطة**: تجنب العناصر الزخرفية الزائدة
3. **الوضوح**: خطوط واضحة وقرائتها سهلة
4. **التفاعل**: ردود فعل فورية للتفاعلات
5. **الثقافة**: دمج العناصر السورية بطريقة احترافية
